I didn't know Arduino is programmed using C. Thankfully a long long time ago when I was about 17 or 18 I took a summer class in C/C++. I was surprised I remembered a few things while looking at some C code.

Using an Arduino Uno, a Grove shield, a Grove dust sensor, and a Grove OLED display I was able to get the data off the dust sensor and display it on the OLED.

I was quite pleased. Looking forward to learning more things using the Arduino...
